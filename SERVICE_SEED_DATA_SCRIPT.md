# Service Seed Data - Quick Add Guide

This document provides sample service data that you can use to quickly add services to your database. The data is formatted for easy use via the admin panel or API.

## How to Use

### Option 1: Add via Admin Panel (Recommended)
1. Log in as admin at `/admin/login`
2. Navigate to Manage Services
3. Click "Add Service"
4. Fill in the form fields using the data below for each service

### Option 2: Add via API (Backend Developer)
Use the data from `SERVICE_SEED_DATA.json` and POST to `/api/admin/services`

## Backend Developer: API Format

If adding via API POST `/api/admin/services`, use this format:

```json
{
  "title": "Water Leakage Repair",
  "description": "...",
  "price": 449,
  "originalPrice": 799,
  "badge": "Visit Within 1 Hour",
  "image": "https://images.unsplash.com/photo-1621905251918-48416bd8575a?w=800&q=80",
  "category": "Water Leakage Repair",
  "process": ["step1", "step2", ...],
  "benefits": ["benefit1", "benefit2", ...],
  "keyFeatures": ["feature1", "feature2", ...],
  "recommendedFrequency": "As needed when leakage occurs"
}
```

See `SERVICE_SEED_DATA.json` for complete JSON format.
